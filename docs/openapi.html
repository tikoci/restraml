RouterOS API Explorer

Interactive explorer for the RouterOS REST API — browse endpoints, view request/response schemas, and generate curl commands. Powered by Scalar and the auto-generated OpenAPI 3.0 schemas. The schema is large (~11 MB, 5000+ endpoints) — initial load takes 20–40 seconds.

About the API Explorer …

Usage: Select a RouterOS version above. The OpenAPI schema for that version loads into the Scalar viewer, where you can:

  • Browse all REST endpoints grouped by RouterOS command path
  • View request body schemas with every available attribute
  • See response schemas and error codes
  • Copy ready-to-use curl commands (or Python, Node.js, etc.)

Authentication: RouterOS uses HTTP Basic Auth. Enter your router’s username (typically admin) and password in the Scalar auth panel. Credentials stay in your browser only.
